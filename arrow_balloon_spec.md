# 弓矢バルーンサバイバル 仕様書

> 引き継ぎ用ドキュメント。セッション間でコアサイクルを軸に開発を進める。

---

## ゲームコンセプト

弓矢を上方向に飛ばし続ける縦スクロールサバイバルゲーム。
風船を割ってスコアを稼ぎ、手持ちの矢が尽きるまで進み続けるハイスコア型ゲーム。

---

## 開発方針

**コアサイクルを先に完成 → その後に詳細・拡張を追加**

### Phase 1（コアサイクル）
1. スワイプで矢を発射
2. 矢が飛んで風船に当たる → スコア加算
3. 木のオブジェクトに刺さる → 次の矢を準備
4. 矢がなくなる → ゲームオーバー

### Phase 2（詳細実装）
- 風船バリエーション（移動型・ワープ）
- ギミック（風）
- UIの完成

### Phase 3（拡張・検討）
- 矢の特殊ギミック（アイテム型 or コンボゲージ型）
- コンボシステム
- 特殊矢の種類

---

## 操作仕様

| 操作 | 動作 |
|------|------|
| スワイプ（引っ張り） | ゴムのように引っ張って離す → 矢が飛ぶ |
| 引っ張り長さ | 飛距離・速度に比例 |
| 引っ張り角度 | 矢の発射方向を決定 |

- 発射地点：現在の矢が刺さっているオブジェクトの原点
- 方向：引っ張った逆方向（弓矢らしい挙動）

---

## 矢（Arrow）の挙動

### 物理・座標
- **原点 = 矢の先端（tip）**
- 飛行中は進行角度に応じて矢が回転表示
- ベクトル方向と矢の向きが一致する

### 壁反射
- 左右端に壁あり
- 矢の原点（先端）が壁に接触 → 反射角で進行方向変更
- 上面：壁なし（無限スクロール）
- 下面：落下判定（画面外に出るとゲームオーバー候補）

### 着地（刺さる）
- 木のオブジェクトに接触すると刺さって停止
- 刺さった状態で次のスワイプ待ち
- 次の矢は「刺さった木オブジェクトの原点」から発射
- 矢を消費する（残り本数 -1）

---

## ライフシステム（矢の本数）

- ゲーム開始時：矢を一定数所持（初期値：5本 ※要調整）
- UI下部に矢を横並びで表示
- 矢が0本になる → **ゲームオーバー**
- 矢の消費タイミング：木に刺さった後、次の矢を使うとき

---

## スコア：風船（Balloon）

### 基本
- 画面上に複数の風船を配置
- 風船に数値（得点）を表示
- 矢が風船に当たると割れる → スコア加算
- 風船には矢マーク（🏹）も表示

### 配置コンセプト
- 隣接配置で連続コンボ感（マリオのコインのような道）
- プレイヤーが通るべきルートが風船の並びで視覚的にわかる

### 風船の種類（Phase 2）

| タイプ | 説明 | 実装優先度 |
|--------|------|-----------|
| 固定 | 動かない | Phase 1 |
| 上下移動 | 上下に往復 | Phase 2 |
| 左右移動 | 左右に往復 | Phase 2 |
| ワープゾーン | 別の場所に瞬間移動 | Phase 2 |

---

## オブジェクト

### 木オブジェクト（着地台）
- 矢が刺さる台
- 木目・丸みのあるデザイン（かわいい系）
- 発射原点として機能

### 壁
- 左右の両端に縦に伸びる壁
- 矢が反射する

---

## ギミック（Phase 2）

| ギミック | 効果 |
|---------|------|
| 風マーク | 矢の方向を強制変更（風の向き・強さで決まる） |

## 画面構成

```
┌─────────────────┐
│  [スコア表示]    │  ← 上部UI
│                 │
│  🎈  🎈  🎈    │  ← 風船（隣接配置）
│         🌳     │  ← 木オブジェクト
│  🎈  🎈       │
│    🌳           │
│  🎈             │
│                 │
│    ↗ 矢        │  ← 矢が飛ぶ
│  🌳（刺さり中） │  ← 現在地点
│                 │
│ [🏹🏹🏹🏹🏹]  │  ← 下部UI：残り矢
└─────────────────┘
```

- 縦スクロール（上方向に進む）
- カメラは矢（現在地点）を追従

---

## ゲームオーバー

- 矢が全てなくなったとき
- （要検討）矢が画面外下に落ちたとき
- ゲームオーバー画面でハイスコア表示

---

## ゲームフロー

```
1. タイトル画面
      ↓
2. ゲーム開始（矢：初期本数, スコア：0）
      ↓
3. スワイプで矢を発射
      ↓
4a. 風船に当たる → スコア加算 → 飛行継続
4b. 木に刺さる → 停止 → 矢消費 → 次スワイプ待ち
4c. 壁に当たる → 反射して飛行継続
      ↓
5. 矢が0本 → ゲームオーバー
      ↓
6. ハイスコア表示 → タイトルへ
```
